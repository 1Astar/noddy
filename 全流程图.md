# Noddy 原型全流程图

> 说明：这是 Mermaid 语法流程图。可在支持 Mermaid 的编辑器中直接渲染，再导出 PNG/SVG。

```mermaid
flowchart TD
  A0[打开应用] --> N0{是否已有 ownerName 和 petName}
  N0 -->|否| N1[进入 welcome 页面]
  N0 -->|是| C1
  N1 --> N2[填写主人称呼 ownerName]
  N2 --> N3[为宠物命名 petName]
  N3 --> C1{WiFi 已完成配置}

  C1 -->|否| W0[进入 WiFi reminder 页面]
  C1 -->|是| H0[进入 home 页面]

  W0 --> W1{是否首次添加设备}
  W1 -->|是| W2[弹出设备选择]
  W1 -->|否| W3[直接去 add device 页面]
  W2 --> W3

  W3 --> W4[进入 WiFi connect 页面]
  W4 --> W5[扫描或手动输入 WiFi]
  W5 --> W6{连接结果}
  W6 -->|成功| W7[写 noddy wifi setup done 等于 1]
  W6 -->|失败| W8[记录失败次数并弹失败提示]
  W8 --> W5
  W8 -->|失败次数小于3| W12[仅显示重试 稍后]
  W8 -->|失败次数达到3| W9[弹窗显示离线登录按钮]
  W9 --> W11[离线登录后直接进入 home 并标记 offline mode]
  W7 --> H0
  W11 --> H0

  H0 --> H1[首页互动 说话 喂食 任务]
  H0 --> H2[点电池 打开电量问答]
  H0 --> H3[用户页 打开完整帮助问答]
  H0 --> H4[点击家长模式入口]
  H0 --> H8[无网络时 顶部横幅提示 部分功能受限 可关闭]
  H4 --> H5[家长验证 算式或长按]
  H5 --> H6[进入家长设置面板]
  H1 --> H7{用户操作跳转条件}
  H7 -->|喂食 任务 番茄钟 社区 消息| D0[跳 detail 页面对应锚点]
  H7 -->|点击 logo 交互日志| D1[跳 detail diary]
  H7 -->|点击成长地图或食物图鉴| D2[跳 detail growth 或 food]
  H3 --> U1[资料信息 昵称 生日]
  H3 --> U2[账号绑定 Google Apple 手机号 邮箱]
  U2 --> U3{用户是否主动绑定}
  U3 -->|否| H0
  U3 -->|是| U4[绑定用于短信通知或提醒功能]
  U2 --> U5[换绑邮箱]
  U5 --> U6{是否已绑定手机号}
  U6 -->|否| U7[提示先做身份验证并绑定手机号]
  U6 -->|是| U8[允许继续换绑邮箱]

  H0 --> L0{是否需要账号能力}
  L0 -->|仅本机体验| H0
  L0 -->|云同步 跨设备 登录保护| A1[进入 signup 页面]

  A1 --> B{选择登录方式}
  A1 --> B0[点击模拟登录失败按钮]
  B0 --> BF[打开登录失败弹窗]
  B -->|Google| G1[Google 授权成功]
  G1 --> G2{邮箱已存在且未绑定 Google}
  G2 -->|是| G3[提示 是否绑定现有账号]
  G3 -->|绑定账号| G4[绑定 Google 到现有账号]
  G3 -->|取消| B
  G2 -->|否| G5[Google 登录成功]
  G4 --> L1
  G5 --> L1

  B -->|Apple| AP1[Apple 授权成功]
  AP1 --> AP2{是否先绑定}
  AP2 -->|绑定邮箱| AP3[进入邮箱登录页验证]
  AP2 -->|绑定手机号| AP4[进入手机登录页验证]
  AP2 -->|稍后绑定| AP5[直接登录]
  AP3 --> AP6[Apple 账号合并到现有账号]
  AP4 --> AP6
  AP6 --> L1
  AP5 --> L1

  B -->|邮箱登录| E1[邮箱密码或邮箱验证码登录]
  E1 --> E2{账号存在}
  E2 -->|否且区域不匹配| E3[提示切换区域]
  E3 -->|切换| E4[跳 index 页面切区并回到 signup]
  E3 -->|继续注册| S1
  E2 -->|是| L1

  B -->|手机号登录| P1[手机号密码或短信验证码登录]
  P1 --> L1

  B -->|注册| S1[注册 输入邮箱验证码和密码]
  S1 --> S2[注册成功 写入账号区域和认证源]
  S2 --> L1
  L1[登录成功] --> H0
```

## 导出方法

1. **最快方式（在线）**
   - 打开 [Mermaid Live Editor](https://mermaid.live/)
   - 粘贴上面的 Mermaid 代码
   - 右上角 `Export` 导出 `PNG` 或 `SVG`

2. **放到文档里**
   - 导出 PNG 后直接插入 Word/飞书/Notion
   - 或者在支持 Mermaid 的 Markdown 工具（Typora、Obsidian）中直接渲染截图

3. **如果你要我直接给你图片文件**
   - 我可以再给你生成一份 `流程图.png` 放在同目录
